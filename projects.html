Projects

AI Heart Attack Prediction

This is an app that can predict the chances of a patient getting a heart attack based on many different factors such as age, gender, and cholestrol. The app uses a random forest algorithm that ranks the importance of the different factor based on a large data set from a real hospital. The algorithm is able to predict with 88% accuracy.
Skills used: Python, Google Colab, Deep Forest, Decision Trees, Bootstrapping, numpy, pandas, matplotlib, sklearn

Alarm Clock Project

This is a program that is able to check the current weather and play music depending on the weather outside. The music is played based on the time that the user decides. The program is run on a Raspberry Pi and has a speaker connected to it to play music.
Skills used: Python, Raspberry Pi, daytime package, Open Weather API, pygame.mixer
